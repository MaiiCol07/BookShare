BOOKSHARE

Narrativa BOOKSHARE es una plataforma en línea para intercambiar libros. Los usuarios pueden registrarse, crear un perfil, y luego publicar los libros que quieren intercambiar o buscar libros que necesiten. Cada usuario tiene un historial de intercambios y puede ver la calificación que recibe de otros usuarios, lo que ayuda a construir una buena reputación en la plataforma.

Los libros se organizan por género, autor, editorial y estado (nuevo, usado, etc.). Los usuarios pueden buscar por estas categorías y enviar solicitudes de intercambio. El sistema muestra si las solicitudes fueron aceptadas, rechazadas o están pendientes.

Después de cada intercambio, ambos usuarios pueden dejar una calificación y un comentario sobre la experiencia. También hay una sección para compartir opiniones sobre el intercambio y la calidad de los libros recibidos.

Descripción del proyecto web

Objetivo: Crear una plataforma que permita a las personas intercambiar libros de manera sencilla y económica.

Público objetivo: Personas que buscan una manera fácil de obtener nuevos libros a través del intercambio.

Funcionalidades:

Registro de usuarios: Crear un perfil y verificar la cuenta.
Publicación de libros: Anunciar libros para intercambio, indicando detalles como título, autor y estado.
Búsqueda de libros: Buscar por género, autor y otros criterios.
Solicitudes de intercambio: Enviar y recibir solicitudes de intercambio.
Calificación de intercambios: Dejar una calificación y comentario después del intercambio.
Historial de intercambios: Ver los libros intercambiados y sus calificaciones.
Tecnologías:

Front-end: HTML y CSS para el diseño del sitio.
Back-end: PHP, Python o JavaScript para el servidor y la gestión de datos.
Base de datos: MySQL o PostgreSQL para almacenar información sobre usuarios, libros y comentarios.
