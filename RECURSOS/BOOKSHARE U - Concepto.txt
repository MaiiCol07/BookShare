	BOOKSHARE U


Narrativa
Una plataforma en línea ofrece servicios de intercambio de libros entre estudiantes universitarios. Los usuarios pueden registrarse en la plataforma y, una vez verificados, pueden publicar libros que desean intercambiar o buscar libros que necesitan. Cada usuario tiene un perfil que contiene su nombre, carrera, universidad y un historial de intercambios. Los libros pueden clasificarse por género, autor, editorial y estado del libro (nuevo, usado, etc.). Los usuarios pueden enviar solicitudes de intercambio a otros usuarios, y la plataforma rastrea si la solicitud fue aceptada, rechazada o está pendiente.

Además, los usuarios pueden calificar el intercambio después de completarlo, y estas calificaciones afectan la reputación del usuario. La plataforma también permite a los usuarios buscar libros por género, autor o universidad, y ver una lista de intercambios recientes que se realizaron con éxito en su universidad.

Por último, la plataforma tiene una sección de comentarios donde los usuarios pueden dejar su opinión sobre la experiencia del intercambio y la calidad del libro recibido.

Descripción del proyecto web
Objetivo:
El objetivo del proyecto web es crear una plataforma de intercambio de libros entre estudiantes universitarios. Los usuarios podrán publicar libros que deseen intercambiar, buscar libros disponibles y contactar a otros usuarios para proponer un intercambio. El sistema gestionará las solicitudes de intercambio y permitirá calificar las transacciones.

Público objetivo:
Estudiantes universitarios que buscan una forma económica de adquirir libros a través de intercambios con otros estudiantes de su misma o diferentes universidades.

Funcionalidades:
Registro de usuarios : El formulario permitirá a los usuarios registrarse, proporcionando su información personal, como nombre, universidad, carrera, etc. Tras el registro, se enviará un correo para verificar la cuenta.

Publicación de libros : Los usuarios podrán crear publicaciones de los libros que deseen intercambiar, indicando detalles como título, autor, género, estado del libro y un breve comentario.

Búsqueda de libros : Los usuarios podrán buscar libros por género, autor o universidad. Los resultados mostrarán detalles del libro y opciones para solicitar un intercambio.

Solicitudes de intercambio : Los usuarios podrán enviar solicitudes de intercambio a otros usuarios, que pueden ser aceptadas o rechazadas. Los estados de las solicitudes se gestionan dentro del sistema.

Calificación de intercambios : Una vez completado el intercambio, ambas partes podrán calificar el intercambio. Estas calificaciones influirán en la reputación del usuario dentro de la plataforma.

Comentarios : Los usuarios podrán dejar comentarios sobre su experiencia de intercambio.

Historial de intercambios : Cada usuario tendrá acceso a su historial de intercambios, donde podrá ver los libros que ha intercambiado y sus calificaciones.

Tecnologías:
Front-end : HTML y CSS para la estructura y diseño del sitio web.
Back-end : PHP, Python o JavaScript para manejar la lógica del servidor, la autenticación de usuarios y las transacciones de la base de datos.
Base de datos : Podrías usar MySQL o PostgreSQL para almacenar la información sobre los usuarios, libros, intercambios y comentarios.
Justificación:
El proyecto ofrece una solución práctica para los estudiantes que buscan una forma económica de acceder a libros, promoviendo la reutilización de recursos y fomentando el intercambio entre estudiantes de diversas universidades.
